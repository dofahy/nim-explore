A very simple ping, using HTTP not ICMP.\
It demonstrates these concepts:
```
1. Concurrency models

  sequential (blocking I/O)
  async (event loop concurrency)
  threads (OS-level parallelism)

2. Networking

  HTTP requests
  DNS resolution
  TLS (HTTPS)

3. Error handling

  unreachable domains
  timeouts
  connection failures

4. Performance comparison

  latency differences between models
```
